Auto Wheelspin Guide

Run Wheelspins Automatically

FAFE can repeatedly spin normal or Super Wheelspins, wait for the collection prompt, collect prizes, and handle duplicate-car menus.

FAFE simulates keyboard and mouse input. Automation may carry account risk; use it at your own discretion.

Choose the mode

  • Select Normal or Super Wheelspin in FAFE.
  • Choose whether duplicate cars should be kept in the garage or sold.
  • Use Sell only when you are comfortable leaving that choice unattended.

What FAFE detects

  1. The Wheelspin tile is selected to start the first spin.
  2. FAFE makes a best-effort attempt to skip the reveal animation.
  3. It waits for the Collect prompt, then presses Enter. A Super Wheelspin collects all three prizes together.
  4. It checks for up to three duplicate-car menus and applies your selected action.

Reliable unattended use

  • Do not minimize the game window; covered or unfocused is fine in background mode.
  • Match the FAFE template language to the game menu language.
  • Keep the solid-yellow ?Car Already Owned? header unobstructed because that is the duplicate detection target.
  • Test duplicate handling before running a large batch.

When collection stops

FAFE intentionally waits when the expected prompt is not detected. Press F9 to stop, inspect the current screen and log, and use F12 on the failed screen if you need a report.
