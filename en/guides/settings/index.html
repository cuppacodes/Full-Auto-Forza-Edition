Part 10 — Settings

Settings

🎨

Appearance

Theme: overall colour scheme (Default / Midnight Neon); switching restarts the app to apply.

Language: UI language (中文/English); switching restarts the app.

Game menu language: which template set to use (Auto / 繁中 / English) — match your game's UI language; Auto follows the app language.

UI scale: app window size (Auto, or 100%–250%); applies live.

Game overlay: show the floating status card over the game (also toggled with F10).

🖥️

System

Mute game while running: mutes the game's audio during a run (per-app, leaves other apps alone).

OCR text detection: use OCR to read on-screen text and confirm matches. Off by default (lighter on CPU, fixes stutter); turning it on helps most when the window is small and the pixel match struggles.

Debug snapshots: save annotated detection PNGs to the debug/ folder (applies next run) for diagnosing mis-detections.

⌨️

Shortcuts

Rebindable: Start/Stop (default F9), Capture (default CAPS LOCK), Report (default F12), Overlay (default F10). Click a button and press any key to rebind.

🏁

Race

Detection interval: how often it scans the game (0.5 = every 0.5s); raise it on a weaker CPU.

Post-key wait: how long to wait after each key press; raise on low FPS or slow loads.

⭐

Unlock Spin Wheel

Cutscene wait: time to wait for the "Ride This Car" cutscene after entering a car.

Node unlock interval: the gap between each unlock (key press) on the mastery tree.

Menu tap interval: the gap between consecutive menu key taps.

🛒

Buy / Delete / Wheelspin

Post-key wait: the key interval for Buy, Delete Cars and Auto Spin Wheel respectively. Drag left to speed up, but too fast may drop inputs or break.

ℹ️

General

Every setting has a ? help button. All settings save instantly (Theme and Language need a restart). If a slower PC misses keys or clicks, raise the matching "wait" value.
