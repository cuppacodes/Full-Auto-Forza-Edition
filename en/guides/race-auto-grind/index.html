Race Auto-Grind Guide

Automate Repeat Races with FAFE

Race Auto-Grind watches the start and results screens, holds the accelerator during the timed race, and restarts the event until you stop it or reach your loop limit.

FAFE simulates keyboard and mouse input. Automation may carry account risk; use it at your own discretion.

Before you start

  • Run Forza Horizon 6 in Borderless or Windowed mode and leave the game window visible?not minimized.
  • Select the correct game monitor in FAFE.
  • If the game runs as administrator, run FAFE as administrator too; Windows otherwise blocks injected input.
  • Leave automatic English IME switching enabled if a Chinese input method may consume Space, Enter, or arrow keys.

How the loop works

  1. FAFE waits for the Start Race screen.
  2. It presses Enter, waits for the countdown, then holds W.
  3. When the results/restart screen is detected, it releases W and selects Restart.
  4. The next loop begins after the start screen returns.

Detection waits indefinitely by design. Press F9 whenever you need to recover or stop safely.

Recommended setup

  • Test one loop while watching the log before leaving it unattended.
  • Use the built-in templates first; they automatically scale to the game resolution.
  • Keep OCR off unless pixel matching struggles in a small game window.
  • Raise Race Detection Interval on a weaker CPU, or raise Post-key Wait if the game misses menu input.

If the race does not move

The usual causes are Windows privilege mismatch, a non-English IME consuming keys, or the game being minimized. Check the FAFE log first. If the problem repeats, stay on the failed screen and press F12 to create a diagnostic report.
