Part 1 — First-time Setup

First-time Setup

1

Pick your game monitor

Select the monitor your game is on from the Game Monitor picker (top-right), and run the game in Borderless or Windowed mode.

2

Templates are built in

Detection templates for Race, Buy and Auto Spin Wheel ship built in and auto-scale to your resolution — nothing to capture, they work out of the box.

3

Appearance & language

In Settings you can switch language (中文/English), theme and UI scale, and rebind hotkeys (default F9 start/stop, F12 report, F10 overlay).

4

Only exception: Unlock Spin Wheel grid

The one thing you set up is Unlock Spin Wheel: in its panel, use the unlock-path grid to mark which mastery-tree nodes to unlock, and the garage block to pick the range of cars (first row, middle full columns, last row). No node capture or templates needed — see Part 3.

5

Ready to go

Everything else works without setup. Open the function you want and press ▶ Start or F9.
