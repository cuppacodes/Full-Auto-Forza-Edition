22B Mastery Guide

Unlock 22B Mastery Nodes in Batches

The mastery automation uses keyboard navigation?not screen templates?to enter each selected car, open Car Mastery, walk your unlock path, and continue through a contiguous garage block.

FAFE simulates keyboard and mouse input. Automation may carry account risk; use it at your own discretion.

Prepare the garage block

  • Sort My Cars by Recently Added.
  • Place the first target 22B-STi at the top-left of the selected range.
  • The selected range must contain only cars you intend to process; navigation is top-to-bottom, column by column, not a snake.
  • You do not need to delete unrelated cars outside the selected block.

Configure the unlock path

In the Mastery grid, mark the nodes FAFE should unlock. The path must be reachable in the same order the keyboard cursor will walk it. FAFE uses WASD and Enter; there is no node screenshot or click-position capture.

Timing settings

  • Cutscene wait covers the ?Ride This Car? transition.
  • Node unlock interval controls the delay between mastery unlock presses.
  • Menu tap interval controls consecutive menu navigation keys.

If a slower PC or low frame rate drops inputs, increase the relevant delay rather than changing the grid.

Protect your garage

Start with a small test range and confirm every car in the selected block is a target 22B-STi. Use F9 immediately if the cursor moves to an unexpected car or menu.
