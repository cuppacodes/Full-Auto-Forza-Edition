About

What is Full Auto Forza Edition?

Full Auto Forza Edition is an automation tool built for Forza Horizon 6 players. Using screenshot template matching and simulated input, it fully automates five of the most time-consuming repetitive tasks — race grinding for skill points, unlocking Spin Wheel rewards from compatible cars' mastery trees, batch-buying any purchasable car, auto-spinning Super or normal Wheelspins (with duplicate-reward handling), and deleting used cars from your garage. It reads the game window directly and sends input via window messages, so it keeps working even when the game is unfocused, covered by another window, or running in the background.

No game file modification, no code injection. The tool simply simulates keyboard and mouse actions, just like a human player would.

Features

Key Features

🏁

Race Auto-Grind

Automatically detects race start, finish and restart screens. Holds W throughout, restarts automatically when the race ends.

⭐

Auto Unlock Spin Wheel from the Mastery Tree

Configure the mastery-tree path for any compatible car and select a garage range. FAFE opens each car's mastery tree, unlocks the chosen nodes, and moves to the next car automatically.

🛒

Auto Buy Cars in Batch

Batch-buys any purchasable car when you open it in Car Collection first. Starting from the main menu uses the automatic 22B-STI preset.

🎡

Auto Spin Wheel

Auto-spins Super or normal Wheelspins from the My Horizon menu — waits for the on-screen prompt, collects the prize (all 3 at once for a Super Wheelspin), and handles duplicate cars (keep or sell) automatically. Loops until stopped.

🗑️

Delete Used Cars

Automatically deletes cars one by one. After each deletion the game auto-selects the next car — no navigation needed. Loops until stopped.

📸

Template Matching

Ships with detection templates that auto-scale to your resolution — Race, Buy and Auto Spin Wheel work out of the box, no capturing needed.

🌏

Two-language UI

Full Traditional Chinese and English support. Language switch auto-restarts the app. All settings save instantly.

⌨️

Customizable Hotkeys

Toggle and capture keys are fully remappable. Default F9 to start/stop, CAPS LOCK to capture. Click a button and press any key to rebind.

🖥️

Multi-monitor Support

Select which monitor your game is on. Supports ultrawide displays. Persists across restarts.

🪟

Background Mode

Reads the game window directly and sends input via window messages, so it runs even when the game is unfocused, covered, or in the background — keep using your PC while it grinds. A keep-alive stops the game auto-pausing while unfocused.

Getting Started

Three Steps to Start

1

Pick your game monitor

Select the monitor your game is on from the Game Monitor picker (top-right). Detection templates ship built in and auto-scale to your resolution — Race, Buy and Auto Spin Wheel need no capturing.

2

Position the Game, Press Start

Place Forza Horizon 6 on the required screen, then return to Full Auto Forza Edition and click "▶ Start" or press F9. The tool counts down 3 seconds. With background mode enabled, you do not need to switch back to the game.

3

Let It Run

The tool runs fully automatically. A live log shows every action in real time. To stop at any time, press F9 or click "■ Stop".
