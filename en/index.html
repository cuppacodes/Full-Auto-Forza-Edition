About

What is Full Auto Forza Edition?

Full Auto Forza Edition is an automation tool built for Forza Horizon 6 players. Using screenshot template matching and simulated input, it fully automates five of the most time-consuming repetitive tasks — race grinding for skill points, unlocking 22B mastery, bulk buying 22B-STi, auto-spinning Super or normal Wheelspins (with duplicate-reward handling), and deleting used cars from your garage. It reads the game window directly and sends input via window messages, so it keeps working even when the game is unfocused, covered by another window, or running in the background.

No game file modification, no code injection. The tool simply simulates keyboard and mouse actions, just like a human player would.

Does the game need to stay focused?

No. FAFE reads the game window directly and sends input via window messages, so it works even when the game is unfocused, covered by another window, or in the background — and a keep-alive stops it auto-pausing. Use your PC for other things while it runs. (Don't minimise the game, though — a minimised window stops rendering frames.)

?

Tool misses a key press or click

Open Settings (⚙) and increase Key Wait or Click Wait. Raise them on slower PCs or when the game loads slowly.

?

Does a resolution change break templates?

No — built-in templates auto-scale to your resolution, so there's nothing to redo. If detection fails at an unusual aspect ratio, report it with F12.

?

Can I use other courses?

Any course you can finish just by holding W works. If the car careens off, crashes or loses control, turn on assisted braking in the game's difficulty settings — that usually keeps it on track. It's the game's auto-steering, not a tool problem.

?

Can it unlock mastery for other cars?

Yes — set up that car's correct unlock path in the unlock-path grid. The tool walks the grid path by keyboard, so it isn't tied to any specific car.

?

Game ignores key presses or clicks (car won't move)

If FAFE is sending input but the game doesn't respond (car won't drive, keys or clicks do nothing), run FAFE as administrator (right-click FAFE.exe → "Run as administrator"). Windows blocks a lower-privilege program from sending input to a game that's running with administrator rights.

🔍

Detection system

Detection works in a few layers: (1) a fast text-appearance match at the pixel level; (2) the ROI system, which limits the search to where the template should appear (e.g. the "Start Race" button is centre-left, so it only searches there) — this also makes detection tens of times faster; (3) optional OCR text hints — the keywords each screen should contain, which OCR reads off the screen to confirm a match. OCR is CPU-heavy and can stutter the game, so it's off by default and can be enabled in Settings (it helps most when the window is small and the pixel match struggles).

There's a single built-in template set that auto-scales to your resolution, so Race, Buy and Auto Spin Wheel work out of the box. Advanced users can capture their own templates and draw a custom Detection area (ROI) for a different game language or custom screen.

📊

Match value (threshold)

Imagine being shown two images and asked how similar they are, as a %. That's the match value — except the system compares at the pixel level, so even if they look identical to your eye, the system might see only 50% similarity. The slider in Settings decides what % similarity counts as a pass. Built-in templates auto-scale to your resolution, so there's no resolution to choose.

🎯

Choosing a template

As a rule, pick a screen element that only appears on that step — e.g. the timer top-left, or "Restart" on the results screen.

Other factors: variability — avoid a translucent background that changes by location; size — a bigger region has more pixels to match (fewer false positives) but is harder to pass and slower to scan, so experiment; language — built-in templates cover Traditional Chinese and English game UI; for other languages or a custom screen, capture your own templates and draw the Detection area.
